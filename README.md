Command line for OneAPM NodeJS agent

[![NPM](https://nodei.co/npm/oneapm-cli.png)](https://nodei.co/npm/oneapl-cli/)


## Features

- Heap Snapshot
- CPU Profile
- Monitoring Dashboard

## Installation

```sh
npm install oneapm-cli -g
```

## Usage

```sh
oneapm YOUR_FILE_NAME
```
